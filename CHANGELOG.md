## [0.1.0] — 2025-02-24

### Added
- Initial public release
- Gateway WebSocket control plane
- Channels: WhatsApp, Telegram, Slack, Discord, SMS, Email, WebChat
- Multi-agent routing with per-session isolation
- Docker sandbox support for group/channel sessions
- Skill system with browser, calendar, email, memory, search, files, cron
- macOS menu bar companion app
- iOS and Android node support
- Voice Wake + Talk Mode via ElevenLabs
- Live Canvas + A2UI
- Tailscale Serve/Funnel integration
- Interactive onboarding wizard (`nexusai onboard`)
- Doctor command for health checks
- In-chat commands: `/status`, `/new`, `/compact`, `/think`, `/model`
- Pairing system for DM security
