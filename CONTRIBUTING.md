# Contributing to NexusAI

Thank you for your interest in contributing! We welcome bug fixes, new features, documentation improvements, and community skill plugins.

## Getting Started

1. **Fork** the repository and clone it locally
2. Install dependencies: `pnpm install`
3. Copy the example config: `cp .env.example .env`
4. Start the dev server: `pnpm gateway:watch`

## Pull Request Guidelines

- Keep PRs focused — one feature or fix per PR
- Add tests for new functionality
- Update documentation if your change affects the public API or configuration
- Follow [Conventional Commits](https://www.conventionalcommits.org/) for commit messages:
  - `feat:` new features
  - `fix:` bug fixes
  - `docs:` documentation changes
  - `chore:` maintenance / tooling

## Reporting Bugs

Open an issue at [github.com/prajitdatta/nexusai/issues](https://github.com/prajitdatta/nexusai/issues) with:
- NexusAI version (`nexusai --version`)
- OS and Node.js version
- Steps to reproduce
- Expected vs. actual behavior

## Community

Join our [Discord](https://discord.gg/nexusai) to discuss ideas before opening big PRs.

AI-generated or AI-assisted PRs are welcome as long as the code is reviewed and tested by the contributor. 🤖
